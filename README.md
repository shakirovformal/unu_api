# Название библиотеки

unu-api

Библиотека поддерживает и реализует все методы доступные API от биржи задач unu.im


## Установка

```bash
go get github.com/shakirovformal/unu_api
```

## Быстрый старт
```golang
 package main

import (
    "fmt"
    "github.com/shakirovformal/unu_api"
)

func main() {
	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	c := api.NewClient(url, token)

	response, err := c.Create_folder(ctx, "сюда пишем имя папки")
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(response.FolderID)
}

```

В библиотеке описаны все входные и выходные параметры взятые из достаточно подробной документации с сайта [text](https://unu.im/api-info)

В некоторых случаях возможно вы будете передавать значения, которые принимают тип "datetime". Для упрощения вашей работы, чтобы вы меньше получали неожиданных результатов, предлагаю вам передавать это значение в виде типа данных string. Пример delay_from:"2025-12-15"
Таким образом, сайт сам преобразует эту дату в нужный для неё вид и всё будет работать корректно. Вы также можете проверить это написав предварительный тест для проверки ожиданий.

Данная библиотека является open-source проектом и доступна для ваших форков и пул реквестов по улучшению кодовой базы
